# GitHub Actionsとは
GitHubが提供するCI/CDサービスです。
[公式のドキュメント](https://docs.github.com/ja/free-pro-team@latest/actions)が充実しているのでこちらを見ていただくのが一番だと思います。

## GitHub Actionsのいいところ
- GitHubとシームレスに利用できる。
- 料金安い。
- トリガーが豊富
- ワークフローファイルを分割できる。
